# AGENT_B_LINUX_NEW - Production Build

## Quick Start

```bash
# Launch the application
./launch_offline.sh

# Or with standard launcher
./launch.sh
```

Access at: http://localhost:7788

## Requirements
- Python 3.10+
- Chrome/Chromium browser
- Virtual environment at: ../turboo_linux_new

## Docker
```bash
docker-compose up -d
```

## Stop
```bash
pkill -f webui.py
```
